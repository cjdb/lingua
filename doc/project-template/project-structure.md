# Project structure

This project is structured so that you can immediately start using it as the basis for your project,
with incredibly minor changes (if any).

```
project_template
├── .git
├── .github
|   └── ISSUE_TEMPLATE
├── cmake
├── config
|   ├── conan
|   |   └── profiles
|   ├── hooks
|   └── scripts
├── include
├── source
└── test
    ├── integration
    ├── regression
    |   └── clang-tidy
    └── unit
```
